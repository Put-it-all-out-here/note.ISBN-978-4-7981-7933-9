## p59
- p59に「MySQL Workbenchで「sakila-schema.sql」を開きます」と記載ある。
  MySQL Workbench上でどう操作すれば「sakila-schema.sql」を開けるのか不明。
  - 対応
    - MySQL Workbenchのトップメニュー:File クリック
    - -> Open SQL Script クリック
    - -> ファイル選択ダイアログで、 「sakila-schema.sql」を選択し「開く」をクリック
